## 2. Paglalarawan ng App

Ang Mei Convertor ay isang lokal, offline-first na utility sa pag-convert ng file na nagpapahintulot sa mga user na:

- Mag-convert ng mga larawan sa pagitan ng mga format (JPEG, PNG, WebP, BMP, atbp.)  
- Mag-convert ng mga larawan at teksto sa PDF  
- Mag-convert ng mga plain-text file sa Word-compatible na mga DOCX na dokumento  
- Kumuha ng teksto mula sa mga DOCX file  
- Pagsamahin at hatiin ang mga PDF na dokumento  
- I-compress ang mga file ng larawan  

Ang lahat ng pagproseso ay ginagawa nang lokal sa device ng user. Walang mga file na ina-upload sa anumang server.
