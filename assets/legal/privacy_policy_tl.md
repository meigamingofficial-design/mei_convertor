# Patakaran sa Privacy — Mei Convertor

**Petsa ng Pagkakabisa:** Hunyo 2026  
**Developer:** Meigaming  
**Kontak:** meigaming.official@gmail.com

---

## 1. Panimula

Ang Mei Convertor ("ang App") ay isang utility para sa pag-convert ng file na binuo ng Meigaming. Ipinapaliwanag ng Patakaran sa Privacy na ito kung paano pinangangasiwaan ng App ang impormasyon na may kaugnayan sa mga gumagamit nito.

Sa pamamagitan ng pag-download o paggamit sa App, kinikilala mo na nabasa at naunawaan mo ang Patakaran sa Privacy na ito.

---

## 2. Impormasyon na Hindi Namin Kinokolekta

Ang Mei Convertor ay idinisenyo bilang isang ganap na **offline-first** na aplikasyon.

- **Walang personal na data na kinokolekta.** Hindi kami nangongolekta ng mga pangalan, email address, numero ng telepono, o anumang mga identifier ng device.
- **Hindi kinakailangan ng user account.** Magagamit mo ang lahat ng tampok ng App nang hindi nagpaparehistro o gumagawa ng account.
- **Walang mga file na ina-upload sa anumang server.** Ang lahat ng mga file na iyong pinoproseso ay nananatili sa iyong device.
- **Walang naka-embed na analytics o tracking services.** Hindi namin sinusubaybayan ang iyong paggamit, gawi, o kasaysayan ng conversion.

---

## 3. Mga Kahilingan sa Pagbura ng Data at Account

Dahil ang Mei Convertor ay hindi nangongolekta, nag-iimbak, o nagpapadala ng anumang data ng gumagamit o nangangailangan ng paglikha ng user account:

- **Walang data sa server-side na maaaring burahin.**
- **Hindi kinakailangan ng mga kahilingan sa pagbura ng account.**
- **Ang lahat ng file ay nakaimbak nang lokal.** Kung nais mong burahin ang mga file na na-convert gamit ang App, maaari mo lamang burahin ang mga output file mula sa lokal na imbakan ng iyong device gamit ang built-in na file manager ng iyong device.

---

## 4. Pagproseso ng File

Ang mga file na iyong pinipili para sa conversion ay:

- Pinoproseso **nang buo sa iyong device** gamit ang lokal na kompyutasyon.
- Hindi kailanman ipinapadala sa, iniimbak sa, o naa-access ng anumang panlabas na server o ikatlong partido.
- Hindi pinapanatili ng App pagkatapos ng conversion; ang mga output file ay nai-save sa lokal na imbakan ng iyong device sa isang landas (path) na maaari mong pamahalaan.

Ikaw ang may tanging pananagutan sa mga file na iyong pinipili at sa mga output na iyong nililikha.

---

## 5. Mga Pahintulot (Permissions)

Maaaring humiling ang App ng mga sumusunod na pahintulot sa device upang paganahin ang pangunahing paggana:

- **Imbakan / Mga File (Storage / Files):** Kinakailangan upang basahin ang mga pinagmulang file at i-save ang mga na-convert na output file sa lokal na imbakan ng iyong device.
- **Media (Android 13+):** Kinakailangan upang ma-access at pumili ng mga file ng larawan para sa conversion.

Ang mga pahintulot na ito ay ginagamit lamang upang magsagawa ng mga lokal na conversion at hindi ginagamit sa anumang iba pang layunin.

---

## 6. Mga Aklatan ng Ikatlong Partido (Third-Party Libraries)

Gumagamit ang App ng mga sumusunod na open-source na aklatan para sa lokal na pagproseso sa mismong device:

- **syncfusion_flutter_pdf** — Pagsasama at paghahati ng PDF (lokal na pinoproseso)
- **archive** — Pagbasa at pagsulat ng ZIP/DOCX (lokal na pinoproseso)
- **flutter_image_compress** — Pag-compress ng larawan (lokal na pinoproseso)

Walang isa man sa mga aklatang ito ang nagpapadala ng data sa labas ng device.

---

## 7. Privacy ng mga Bata

Ang App ay hindi nakadirekta sa mga batang mababa sa edad na **13**. Hindi kami sadyang nangongolekta ng personal na impormasyon mula sa mga bata. Kung naniniwala ka na ang isang bata ay nagbigay ng impormasyon sa pamamagitan ng App, mangyaring makipag-ugnayan sa amin sa meigaming.official@gmail.com upang matugunan namin ang alalahanin.

---

## 8. Seguridad ng Data

Dahil ang lahat ng pagproseso ng file ay nangyayari sa device at walang data na ipinapadala sa labas, ang pangunahing responsibilidad sa seguridad ay nakasalalay sa operating system ng iyong device at sa iyong sariling mga kasanayan sa pamamahala ng file. Inirerekomenda namin na:

- Panatilihing updated ang OS ng iyong device at ang mga security patch nito.
- Gamitin ang lock screen ng device at ang mga tampok ng encryption na ibinigay ng iyong operating system.

---

## 9. Patalastas (Advertising)

Ang bersyong ito ng App ay **hindi nagpapakita ng mga patalastas**. Kung sakaling maglagay ng patalastas sa mga susunod na update, ang Patakaran sa Privacy na ito ay ia-update bago maging live ang mga naturang tampok, at ang mga gumagamit ay aabisuhan sa pamamagitan ng listahan sa app store.

---

## 10. Mga Pagbabago sa Patakarang Ito

Maaari naming i-update ang Patakaran sa Privacy na ito paminsan-minsan. Ang anumang mga pagbabago ay:

- Masasalamin sa pamamagitan ng updated na **Petsa ng Pagkakabisa** sa itaas ng dokumentong ito.
- Ilalathala sa loob ng listahan sa app store para sa App.

Ang patuloy na paggamit ng App pagkatapos ng isang update sa patakaran ay nagpapahiwatig ng pagtanggap sa binagong patakaran.
