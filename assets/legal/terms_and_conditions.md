## 2. Description of the App

Mei Convertor is a local, offline-first file conversion utility that allows users to:

- Convert images between formats (JPEG, PNG, WebP, BMP, etc.)  
- Convert images and text to PDF  
- Convert plain-text files to Word-compatible DOCX documents  
- Extract text from DOCX files  
- Merge and split PDF documents  
- Compress image files

All processing is performed locally on the user's device. No files are uploaded to any server.
