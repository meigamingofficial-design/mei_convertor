1. Introduction

Mei Convertor ("the App") is a file conversion utility developed by Meigaming. This Privacy Policy explains how the App handles information related to its users.

By downloading or using the App, you acknowledge that you have read and understood this Privacy Policy.

2. Information We Do Not Collect

Mei Convertor is designed as a fully offline-first application.

  • No personal data is collected: We do not collect names, email addresses, phone numbers, or any device identifiers.
  • No user accounts are required: You can use all features of the App without signing up or creating an account.
  • No files are uploaded to any server: All files you process remain locally on your device.
  • No analytics or tracking: We do not track your usage, behavior, or conversion history.

3. Data Deletion and Account Deletion Requests

Because Mei Convertor does not collect, store, or transmit any user data or require user account creation:

  • No server-side data exists to be deleted.
  • No account deletion requests are necessary.
  • All files are stored locally: If you wish to delete files converted using the App, you can simply delete the output files from your device's local storage using your device's built-in file manager.

4. File Processing

Files you select for conversion are:

  • Processed entirely on your device using local computation.
  • Never transmitted to, stored on, or accessible by any external server or third party.
  • Not retained by the App after conversion; output files are saved to your device's local storage at a path you can manage.

You are solely responsible for the files you select and the outputs you generate.

5. Permissions

The App may request the following device permissions to enable core functionality:

  • Storage / Files: Required to read source files and save converted output files to your device's local storage.
  • Media (Android 13+): Required to access and select image files for conversion.

These permissions are used solely to perform local conversions and are not utilized for any other purpose.

6. Third-Party Libraries

The App uses the following open-source libraries for local, on-device processing:

  • syncfusion_flutter_pdf — PDF merge and split (processed locally)
  • archive — ZIP/DOCX reading and writing (processed locally)
  • flutter_image_compress — Image compression (processed locally)

None of these libraries transmit data externally.

7. Children's Privacy

The App is not directed at children under the age of 13. We do not knowingly collect personal information from children. If you believe a child has provided information through the App, please contact us at meigaming.official@gmail.com so we can address the concern.

8. GDPR & CCPA Compliance

Because Mei Convertor does not collect, process, or store any personal data, the App is inherently compliant with the EU General Data Protection Regulation (GDPR) and the California Consumer Privacy Act (CCPA).

  • No data subject access requests are required, as no personal data is held.
  • No "Do Not Sell My Personal Information" mechanism is required, as no data is sold or shared.
  • No consent management or cookie banners are needed, as no tracking occurs.

9. Data Security

Because all file processing occurs on-device and no data is transmitted externally, the primary security responsibility rests with your device's operating system and your own file management practices. We recommend:

  • Keeping your device OS and security patches up to date.
  • Using device lock screens and encryption features provided by your operating system.

10. Advertising

This version of the App does not display advertisements. If advertising is introduced in a future update, this Privacy Policy will be updated before such features go live, and users will be informed via the app store listing.

  • Reflected with an updated Effective Date at the top of this document.

Continued use of the App after a policy update constitutes acceptance of the revised policy.
